This directory is a task-scoped asset namespace for the `inhand_transfer` scene.

The task randomizer samples the categories listed in `INHAND_OBJECT_CATEGORIES`
from this folder instead of depending on the full ignored RoboCasa asset cache.

Copied source categories:

- `dish_brush`, `whisk`, `salt_and_pepper_shaker`, `cream_cheese_stick`,
  `cheese_grater`, and `pizza_cutter` from
  `assets_robocasa/objects_lightwheel/lightwheel/`
- `rolling_pin`, `water_bottle`, `can`, and `ladle` from
  `assets_robocasa/objaverse/objaverse/`
